About Hyparkour

Hyparkour is a site which allows you to see player's best parkour times and compare them to the world records.

The data is based on speedrun.com's API, so as soon as a new run is verified it will be automatically updated on this site.

Hyparkour wouldn't have been possible without the following:

- speedrun.com's API for all the record times
- mc-heads.net's API for player heads and fallback name capitalization
- Hypixel's API for looking up individual player's times

The parkour screenshots were taken by me and are freely available for reuse and modification on GitHub.
If any issues emerge, please report them to me by using GitHub Issues.
